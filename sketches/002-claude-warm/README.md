## Variant: Claude 暖色风格

### Design stance
受 Claude AI (Anthropic) 启发的温暖编辑风格。羊皮纸底色 (`#f5f4ed`) + 陶土红 (`#c96442`) 强调色，温暖亲切的 AI 伴侣感。

### Key choices
- **布局**: 单栏居中，640px 宽，章节式呼吸间距
- **配色**: 暖米白背景 + 暖灰卡片 + 陶土红强调色
- **字体**: 标题使用 Noto Serif SC（思源宋体）增加文字感和权威感
- **场景选择**: 白色卡片网格，选中态用陶土红边框 + 光晕
- **输入区**: 暖白卡片，圆角 12px，focus 时陶土红高亮
- **角色选择**: 4 列网格，圆角卡片样式
- **提交按钮**: 陶土红实心按钮，hover 渐变亮
- **进度条**: 陶土红渐变填充
- **细节**: 所有圆角温和（10-16px），字体行距偏宽松 (1.6)

### Trade-offs
- **Strong at**: 视觉温暖友好，阅读体验舒适，中文字体展现好
- **Weak at**: 不如暗色主题"科技感"强烈
- **Neutral**: 更适合内容创作类工具

### Best for
喜欢温暖编辑风格、追求像 Claude 一样有"人情味" AI 感的设计。参考了 Anthropic Claude、Notion 等产品的视觉语言。
