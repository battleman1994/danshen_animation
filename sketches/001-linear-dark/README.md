## Variant: Linear 暗色风格

### Design stance
受 Linear.app / Cursor / Vercel 等 AI 工具启发的深色极简风格。黑底 (`#08090a`) + 品牌靛紫 (`#5e6ad2`) 强调色，展现科技感与专业感。

### Key choices
- **布局**: 单栏居中，640px 宽，呼吸式间距
- **配色**: 暗色背景 + 透明微白边框 + 靛紫单一强调色
- **场景选择**: 用卡片式网格，选中态用靛紫边框 + 微光效
- **输入区**: 透明背景 + focus 时靛紫边框高亮
- **角色选择**: 4 列网格，轻量卡牌样式
- **提交按钮**: 靛紫实心按钮，hover 渐变亮
- **进度条**: 靛紫渐变填充 + 状态描述文字
- **字号**: Inter 字体，510 强调权重，紧凑跟踪

### Trade-offs
- **Strong at**: 科技感强，视觉干净专注，符合现代 AI 工具审美趋势
- **Weak at**: 暗色主题在某些环境（强光）下可读性稍弱
- **Neutral**: 需要用户适应暗色 UI

### Best for
喜欢深色模式、追求现代 AI 工具风格的用户。参考了 Linear、Cursor、Replicate 等产品的视觉语言。
